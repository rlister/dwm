* My dwm hacks

Guide here: https://dwm.suckless.org/customisation/patches_in_git/

** Updates

   #+BEGIN_SRC
   git checkout master
   git pull -v --all
   git checkout hacks
   git rebase --preserve-merges master
   ... fix merge conflicts ...
   git rebase --continue
   git push origin
   #+END_SRC


** Install

   #+BEGIN_SRC
   git checkout hacks
   make
   sudo make install
   #+END_SRC
